# America's Founder Class as the Strategic Offset — Exhibit Data & Code

Reproducible code and data behind the three exhibits in the Cardinal40 economist exercise.
The thesis: in the US–China critical-technology contest, America's entrepreneurial **founder
ecosystem** is its hardest-to-copy advantage. Each exhibit is generated by a single Python
script that reads from `data/` and writes a PNG to `figures/`.

**The point of this repo is traceability.** Every headline number on every exhibit can be traced
to a row in a CSV in `data/`, and every CSV row carries its source. Narrative provenance lives in
`SOURCES.md` (Exhibits 1 & 3) and in `SOURCES_Q2.md` + `data/q2_calculations.csv` (Exhibit 2).

## Reproduce

```bash
python -m venv venv && source venv/bin/activate     # optional
pip install -r requirements.txt
python make_exhibits.py                              # builds all three
# ...or run an individual exhibit:
python plot_exhibit1.py
```

Python 3.9+. The scripts resolve paths relative to themselves, so they run from any working
directory. **No network access is needed to render the exhibits** — every input is committed to
`data/`. Rendering all three takes a few seconds.

## Exhibit 1 — "The US Has Fallen Behind in Critical-Tech Research"

- **Script:** `plot_exhibit1.py` · **Inputs:** `data/aspi_tech_flip.csv`,
  `data/aspi_categories.csv`, `data/aspi_us_led_gap.csv`
- **Source:** ASPI Critical Technology Tracker (two-decade report, Aug 2024).
- **How the numbers are derived:**
  - The **3 → 57** (China) and **60 → 7** (US) "technologies led" flip is read directly from
    `aspi_tech_flip.csv` (`technologies_led`, periods 2003–2007 and 2019–2023).
  - The **8.6-pt** thin-lead figure is computed in-script as
    `mean(us_share) − mean(china_share)` over the seven US-led technologies in
    `aspi_us_led_gap.csv` (28.8 − 20.2).
  - The **13% / 39%** high-impact-research shares are 2019–2023 figures derived from ASPI
    Appendix 1 — see methodology note (2) below; labelled indicative on the exhibit.
  - Panel-B category lists are the top three of each group in `aspi_categories.csv`.

## Exhibit 2 — "While China Leads in Inputs, America is Better at Building Companies"

- **Script:** `plot_exhibit2.py` · **Inputs:** `data/q2_calculations.csv`
- **Every bar is a ratio read from `q2_calculations.csv`.** That file carries, per row: the US
  value, the China value, the **explicit formula**, the resulting multiple, the leader, a caveat
  flag, and the source. The script parses the multiple (e.g. `9.18 -> 9.2x`) and the leader from
  that file — nothing is typed into the chart. Change a number in the CSV and the bar moves.
- **Normalization:** ratios are per-capita or per-GDP where noted (VC as % of GDP, unicorns per
  million people, market cap as % of GDP, etc.), so the comparison is size-adjusted, not raw count.
- **Sources** (per CSV row): CSET; World Bank; UN WPP; KPMG/CB Insights; Hurun; Damodaran/CEIC;
  World Bank Doing Business 2020.

## Exhibit 3 — "Backing American Founders Pays Twice"

- **Script:** `plot_exhibit3.py` · **Inputs:** `data/exhibit3_returns.csv`,
  `data/sp500_founder_built_weights.csv`, `data/exhibit3_societal.csv`
- **How the numbers are derived:**
  - **2.1×** total shareholder return (founder-led vs non-founder peers, since 2015; 2.6× in
    tech) — `exhibit3_returns.csv` (Bain & Company).
  - **33%** of the S&P 500 is founder-built — **computed in-script as the sum of the published
    index weights** of the seven founder-built top-10 companies in
    `sp500_founder_built_weights.csv` (7.90 + 6.78 + 6.03 + 4.37 + 3.71 + 1.96 + 1.75 = 32.5 ≈ 33%).
    A reviewer can re-sum the `index_weight_pct` column. The seven company names on the exhibit
    are also read from this file.
  - **31%** more patents (Purdue / HBR) and **~3M** net new jobs/yr (Kauffman / U.S. Census BDS) —
    `exhibit3_societal.csv`.
- **Color note:** navy/red are distributed as a checkerboard (each column carries one navy and one
  red stat) so neither side reads as single-colored.

## Key methodological choices (and honest caveats)

1. **"Leads" = research output, not deployed capability.** Exhibit 1's ASPI metric is share of the
   most-cited research papers — a *leading indicator* of future capability, not fielded systems.
   Stated on the exhibit.
2. **The 13% / 39% research shares are derived from ASPI Appendix 1, not a single printed stat,**
   and are treated as indicative — labelled as such on the exhibit and in `SOURCES.md`.
3. **Exhibit 2 is size-adjusted.** Where China leads on raw counts (e.g. total firm formation), the
   exhibit says so; the bars use per-capita / per-GDP ratios to avoid a population artifact. The
   China VC figure (~$35B) behind the 3.8× funding ratio is an **estimate**, flagged in
   `q2_calculations.csv`.
4. **Exhibit 3 mixes two units, and says so.** Returns and patents describe *founder-led public
   companies*; the jobs figure describes *startups / young firms* (the broader founder ecosystem).
   The exhibit footer and `SOURCES.md` keep this distinction visible rather than implying that
   founder-led mega-caps create all the jobs.
5. **Insolvency-recovery / Doing Business data (Exhibit 2) is 2019-vintage** (the series was
   discontinued); used for the directional "second chances" point and flagged.

## Optional: refresh the open-API series

`scripts/pull_api_sources.py` re-pulls the fully-open, machine-readable series (World Bank WDI,
UN WPP, IMF COFER) used in the supporting research and writes full-series CSVs to `data/`.
Endpoints were verified June 2026 but the script was **not executed in the build environment**
(no network egress), so the committed `*_checkpoints.csv` files record the values the pulled
series should match. Run locally with `requests` installed:

```bash
cd scripts && python pull_api_sources.py
```

These series feed the supporting essay (Q1/Q3), not the three exhibit scripts above, which run
entirely from the committed CSVs.
